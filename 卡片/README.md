## 卡片/



原子化知识卡片——平时学习过程中随手记录的、自包含的小内容。每张卡片独立成篇，记录一个具体知识点或踩坑经验。

卡片不同于系统性笔记：卡片是"一个问题 + 一个答案"的精简格式，篇幅短小，方便快速回顾和按关键词检索。



### 子目录结构

| 子目录 | 当前 | 预估上限 | 结构策略 |
|--------|------|----------|----------|
| （无） | 6 | ~50 | `NN-title.md` 扁平编号，≥50 后按 `01-10/` 范围分组 |

### 命名格式

`NN-title.md` — 两位递增数字 + 中文/中英混合简短标题。新卡片取当前最大编号 +1。

### 内容格式

短卡片（推荐）：问题/解决方案二元结构，简单直接，无需标题层级。
长卡片：可使用 `##` 文档标题 + `###` 内容标题，允许 YAML frontmatter（`create-time`、`update-time`）。
