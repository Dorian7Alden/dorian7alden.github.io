### Daily English Dictation/

每日英语听写练习，按难度分四级：beginner（入门）→ intermedia（中级）→ advanced（高级）→ expert（专家）。

**结构设计**：75 篇超过 50 篇阈值，已按 `001-010/`、`011-020/` 等范围分组，每组 10 篇。后续从 081 继续编号，每满 10 篇新增一个范围目录。

**根目录元文件**（6 个）：
- `template.md` — 听写笔记模板（Listening Practice + Accumulation）
- `0-levels.md` — 难度等级对照表
- `1-blue-beginner.md` ~ `4-red-expert.md` — 各级别说明（颜色标签）

**命名待修复**：
- `Dictation-XXX.md` → `XXX-level.md`（统一去掉 Dictation- 前缀）
- `-hard-hard` → `-hard`，`advance` → `advanced`
