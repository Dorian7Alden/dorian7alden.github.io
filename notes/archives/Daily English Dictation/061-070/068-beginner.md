## Daily English Dictation 68 - beginner



### 🎧 Listening Practice



**My listening**: 

```
Stop it! Stop what? You're talking about me in Morse code. You know when jokes on you in Morse code. Hah. 
```

**Answer**: 

```
Stop it! Stop what? You're talking about me in Morse code. But you know what? Joke's on you 'cuz I know Morse code.
```

**Pronunciation**:

Stop it! Stop what? You're talking ~~a~~bout me in Morse code. But you know what? Joke's on you 'cuz I know Morse code.

> you're => yer
>
> taking about => tawking 'bout
