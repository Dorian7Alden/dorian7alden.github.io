## 日常学习卡片/



连续打卡学习的每日记录——按月份和星期组织，每天输出一篇学习日志，记录当天学到的所有内容。

> 注意：此目录内容尚未原子化，后续计划拆分为独立的知识碎片后再重新分类。
> 当前保留原始结构：`{月份}/week{N}/day{N}.md`



### 子目录结构

```
日常学习卡片/
├── 26-03/          ← 2026年3月
│   ├── week1/ (2)
│   ├── week2/ (6)  ← 含 prompt.md（AI生成的Java后端学习计划）
│   └── week3/ (5)
└── 26-04/          ← 2026年4月
    ├── week1/ (1)
    ├── week2/ (4)
    └── week3/ (2)
```
