**什么是 HC ？**

HC 是 **Head Count** 的缩写，中文直译是「人员编制」，在互联网招聘（校招 / 实习）语境里，**直白说就是「招聘名额 / 岗位坑位」**，是你投递前必须关注的核心信息。

- 这个部门还有 HC 吗 ? 
- 「HC 冻结」= 公司 / 部门暂停招聘，关闭了招聘名额，不再接收新投递；
- 「锁 HC」= 这个岗位的名额已经被候选人占用，进入 offer 审批环节，不再开放新投递。

**什么是 ToB ? ToC ?**

- ToC (To Consumer) : 面向消费者。直接面向**个人用户**提供产品或服务。抖音、淘宝、微信...
- ToB (To Business) : 面向企业。为**企业、组织或商家**提供解决方案、工具或服务。内部办公系统、企业微信后台、商家管理系统...

#### 算法题

- 昨天的栈简单题做法太巧了：[有效的括号](https://leetcode.cn/problems/valid-parentheses/)
  - 遍历方式并不总是要for(int i; ;++i)，有些时候只对每个元素处理时，可以for (auto it: arr)，减少寻址次数


```c++
bool isValid(string s) {
    stack<char> sk;

    for (char ch: s) {
        // 遇到左括号，压栈对应的右括号，栈自动记录了层级结构
        if (ch == '{') sk.push('}');
        else if (ch == '(') sk.push(')');
        else if (ch == '[') sk.push(']');
        // 栈为空说明这个左括号之前没有右括号
        // 不是左括号，该右括号应该与上个左括号所需的右括号相等
        else if (sk.empty() || sk.top() != ch) return false;
        else sk.pop();
    }
    // 左括号还有剩余说明没闭合完
    return sk.empty();
}
```

```java
public boolean isValid(String s) {
    while(true) { // 神人做法 - 简单粗暴
        int l=s.length();
        s=s.replace("()","");
        s=s.replace("{}","");
        s=s.replace("[]","");
        if(s.length()==l){return l==0;}
    }
}
```
