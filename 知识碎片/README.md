## 知识碎片/



原子化知识片段——平时学习过程中随手记录的、自包含的零散内容。每篇独立成篇，篇幅短小，方便按关键词快速检索。



### 子目录结构

| 子目录 | 当前 | 预估上限 | 结构策略 |
|--------|------|----------|----------|
| （无） | 6 | ~50 | `NN-title.md` 扁平编号，≥50 后按 `01-10/` 范围分组 |

### 命名格式

`NN-title.md` — 两位递增数字 + 中文描述性标题。新碎片取当前最大编号 +1。

### 内容格式

短碎片（推荐）：问题/解决方案二元结构，简洁直接。
长碎片：可使用 `##` 文档标题 + `###` 内容标题。
