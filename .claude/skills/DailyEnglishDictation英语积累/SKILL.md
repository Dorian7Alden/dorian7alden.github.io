## 任务背景
用户进行 Daily English Dictation 听写练习，每天完成一个句子的听写与发音笔记，但对句中的词汇、短语、文化背景缺乏系统积累。本 Skill 用于从句子中自动提炼知识点，生成结构化的积累卡片，方便用户直接粘贴到日常笔记。

## 适用场景
- 用户提供一个英语句子（含听写结果及正确答案），要求提取并分类积累其中语言点。

## 核心目标
按知识点类别依次提取并输出：
1. 核心词汇（含释义、搭配、例句、同义替换）
2. 短语与固定表达
3. 特殊句式与语法结构
4. 发音现象（连读、弱读、省音、重音等）
5. 易错点提醒（听写、拼写、理解偏差）

## 输入格式
用户输入应包含：
- 听写内容（用户写下的版本，可选）
- 正确答案（完整英语句子）
- 额外要求（如指定输出路径或特定聚焦点）

## 执行流程
1. **确认目标句子**：以用户提供的“正确答案”为准。
2. **知识点提取**：
   - 识别并归类词汇（B2及以上重点词汇）、短语、语法结构、发音特征。
   - 若提供听写内容，对比正确答案标注典型错误点。
3. **卡片生成**：
   - 每个知识点按原子卡片格式输出（词汇卡/短语卡/语法卡/发音卡/易错卡）。
   - 内容包含：核心内容、释义/说明、典型例句、同义表达/对比（若适用）。
4. **输出与落盘**：
   - 以 Markdown 格式返回卡片集合。
   - 若需写入文件，遵循下方写入规则。

## 写入规则
- **默认路径**：`/English Learning/Daily English Dictation/{yyyy-MM-dd}.md`
- **文件存在**：在文件末尾追加内容（添加日期标题分隔）。
- **文件不存在**：新建文件并写入。
- **用户指定路径优先**。

## 质量检查（生成后自检）
- [ ] 词汇释义准确，例句地道且与语境相关
- [ ] 发音现象描述清晰，附带示例对比
- [ ] 易错点明确，有针对性纠正建议
- [ ] 卡片分类正确，无重叠遗漏
- [ ] Markdown 格式规范，便于笔记软件渲染

## 输出要求（对用户的最终回复）
- 简述已生成卡片数量及类型分布
- 写入的文件路径（若落盘）
- 若某类知识点无内容，说明跳过原因

> 仅在用户明确要求时，才展开显示完整卡片内容。
