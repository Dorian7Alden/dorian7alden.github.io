# **A：测试时间：楼梯的持续磨损**

石头是坚定不移永恒的象征，雕刻过的岩石被用作建筑材料，是因为它能够抵抗磨损。尽管石头具有耐久性，但它并非不受磨损。比石头更坚韧的是人类的持久性。

图1：长期使用后台阶出现不均匀磨损的示例。

用于建造台阶的石头和其他材料会受到持续的长期磨损，这种磨损可能是不均匀的。例如，非常古老的寺庙和教堂可能有台阶，其中台阶的中心部分比边缘磨损得更严重，踏步不再有平整的顶部，而是看起来呈弯曲状。由于这类建筑的性质，这些建筑往往被人们长期居住，但特定地点的人类活动时间往往早于建筑的建造时间。这使得准确确定建筑的建造时间变得困难。当建筑的建造时间跨度很长，进行了翻新，并且随着时间的推移增加了新部分时，情况会变得更加复杂。你的团队被要求提供指导，说明考古学家可以从一组磨损的台阶中确定哪些信息。台阶可能由各种材料建造，例如石头或木材。此外，可能可以估计一组台阶的建造时间，但很难确定精确时间。从历史记录中可能不清楚建筑中哪组台阶是在每个时间段建造的。除了年龄之外，考古学家可能还对与楼梯使用相关的交通模式感兴趣。例如，人们是同时上下楼梯，还是在任何给定时间主要使用一个方向？考古学家还可能想知道楼梯的使用频率。例如，是大量人在短时间内使用它，还是少数人在很长一段时间内使用它？你的团队被要求开发一个模型，可以根据一组台阶的磨损模式得出一些基本结论：

- 楼梯的**使用频率**是多少？
- 使用楼梯的人是否**倾向**于某个**方向**的行走？
- 同时**使用**楼梯的**人数**有多少？（例如，人们是否**并排爬楼梯**，还是**单列行走**？）

你可以假设考古学家可以进入相关建筑，并可以获取你的团队认为重要的任何测量数据。测量必须以非破坏性的方式进行，成本必须相对较低，并且可以由少数人使用最少的工具进行测量。你应该清楚说明所需的测量内容。还有其他一些问题可能更难以解决。假设存在对楼梯年龄的估计，楼梯的使用方式，以及建筑中日常生活的模式，确定可以为以下问题提供哪些指导：

- 磨损是否与可用信息一致？
- 楼梯井的年龄是多少，估计的可靠性如何？
- 进行了哪些修复或翻新？
- 可以确定材料的来源吗？例如，如果使用石头，磨损是否与考古学家认为的原始采石场的材料一致，或者如果使用木材，磨损是否与假设使用的树木的年龄和类型一致？

美国阿拉斯加州朱诺市（Juneau）人口约为3万名居民。2023年，该市创下了接待160万邮轮乘客的纪录，在最繁忙的日子里，最多有七艘大型邮轮停靠，游客数量高达约2万人。虽然这些游客为城市带来了可观的收入，大约3.75亿美元，但同时也带来了过度拥挤的问题，导致城市不得不设法限制游客数量。讽刺的是，朱诺的主要景点之一门登霍尔冰川（Mendenhall Glacier）由于部分由过度旅游导致的气温升高而正在消退。自2007年以来，冰川已经后退了相当于八个足球场的距离，许多当地人担心，随着冰川的消失，游客和相关收入最终也会随之消失。幸运的是，朱诺还有其他景点，包括观鲸和雨林，只要能够制定并实施可持续旅游计划，就可以保持其作为旅游目的地的地位。

# **D：打造更美好城市的路线图**

#### **背景**

交通系统对城市的增长和居民生活有深远影响。成功的交通基础设施能够吸引商业、学校、游客和新居民。然而，城市面临的交通挑战复杂且相互交织。不同的利益相关者（城市居民、企业主、郊区居民、通勤者、过境旅行者、游客等）在交通系统中有不同的需求和优先级。例如，高速公路、公交线路和铁路系统可能干扰城市自行车手和行人，反之亦然。城市的地理环境（如水域、地形、土壤成分和天气条件）也可能对交通系统造成阻碍。

#### **问题背景：巴尔的摩市**

巴尔的摩是美国马里兰州的一座城市，其交通基础设施老化，交通选择有限，影响了居民生活和经济发展。最近，该市一座主要桥梁（弗朗西斯·斯科特·基伊桥）坍塌，关闭了穿过繁忙港口的主要高速公路。巴尔的摩计划通过改善基础设施和增强公共交通来实现可持续发展目标，这些目标包括**利益相关者之间的合作**、**维护或更新物理系统**、**更有效地利用数据**以及**寻求技术进步**。

**情境**：一片森林曾经高耸入云，拥有丰富的野生动植物，但为了农业开发被砍伐殆尽。曾经繁荣的生态系统——鸟类、昆虫和动物的家园——消失了，取而代之的是一排排的农作物。土地开始发生变化——曾经富含自然养分的土壤变得贫瘠，害虫开始入侵农作物。为了对抗这一问题，农民转向使用化学品，但土地的平衡被打破。随着这一转变，曾经在森林中繁荣的复杂生命网络被打破，取而代之的是一个新的、由人类驱动的农业循环，形成了一个新的基于农业生态系统的食物网。在一个成熟的农业系统中，存在着蝙蝠、鸟类和其他物种，但要达到这种状态，生态系统必须成熟。**建模与分析**：在世界各地，类似的情景不断发生。作为“成熟农业实践考虑小组”（COMAP）的成员，你被要求构建一个模型，追踪从森林到农场的栖息地变化。你的主管让你的团队负责确定一个被改造的森林区域如何随着时间的推移而变化，同时生态系统随着农业选择的演变而演变。你的主管希望分析既包括自然过程，也包括人类决策。因此，你应该从一个新清理的改造森林区域生态系统开始建模，并追踪模型因物种变化以及农业实践的许多影响而经历的变化阶段。你可以假设一个从森林到农场的情境，或者使用这种演变阶段的真实历史样本的数据和信息。你可能希望在分析中考虑以下内容：

### 自然过程

- **建模当前生态系统**：为这个新农业生态系统构建一个基本的食物网模型，它最近取代了一个森林密集的区域。包括生产者和消费者，以及农业周期及其季节性对系统动态随时间变化的影响。考虑除草剂和杀虫剂的影响，包括化学品使用对植物健康、昆虫种群、蝙蝠和鸟类种群以及生态系统稳定性的影响。
- **物种的重新出现**：随着时间的推移，边缘栖息地开始成熟，这使得本地物种重新回归。随着物种的回归，农业生态系统因这些物种与当前环境的相互作用而发生变化。将两种不同的物种纳入模型，以确定其影响。

### 人类决策

- **除草剂的去除**：随着生态系统的成熟，农民可能会尝试减少对化学品的依赖。
  - 如果去除除草剂，报告生态系统在生产者和消费者方面的稳定性。
  - 通过将蝙蝠纳入食物网模型，使生态系统恢复平衡。将蝙蝠建模为控制害虫种群的食虫动物，以及支持植物繁殖的传粉者。考虑蝙蝠与昆虫、植物和捕食者的相互作用如何影响整个生态系统的稳定性。确定另一种可以带来平衡的物种，并比较其影响。
- **转向有机农业？**：分析农民考虑有机农业方法的含义。应考虑不同有机农业组成部分的不同情景。展示对整个生态系统以及各个组成部分的影响。讨论害虫控制、作物健康、植物繁殖、生物多样性、长期可持续性和成本效益等方面。

**术语表**

- **改造森林区域**：一片森林被清理以便用于农业的土地。
- **食物网**：食物网是一个生态系统中生物之间复杂的捕食关系网络。它描绘了能量和营养如何在生态系统中从生产者（植物）到多个消费者层级（食草动物、食肉动物和杂食动物）再到分解者（真菌、细菌）流动和相互作用。它对于理解生态系统如何运作和维持平衡至关重要。
  - **初级生产者**：植物和农作物。
  - **初级消费者**：以植物为食的食草动物。
  - **次级消费者**：以食草动物为食的食虫动物和其他捕食者。
  - **三级消费者**：以次级消费者为食的大型捕食者。
  - **分解者**：分解死亡生物并将营养返回生态系统的真菌、细菌和蚯蚓。
- **农业生态系统**：农业生态系统是复杂的，各种物种在食物网中相互作用，既支持生态平衡，也支持农作物生产。人类干预，如使用除草剂、杀虫剂和化肥，通常被用来提高农作物产量，但这些可能会带来负面的生态和经济后果。蝙蝠提供了有价值生态系统服务，如害虫控制和授粉，有助于农业的可持续性。
- **农业周期**：农业周期指的是从种植和生长农作物到收获并准备用于消费或销售的农业生产的各个阶段。关键阶段包括：
  - **土壤准备**：农民通过耕作、除草和添加化肥来准备土地。土壤至关重要，因为它作为农作物生长的介质，支持食物网。
  - **种植/播种**：农民在适合农作物生长的时间种植种子或幼苗。这取决于天气、温度和土壤条件。
  - **生长阶段**：农作物通过光合作用生长，它们从空气中吸收二氧化碳，从土壤中吸收水分，并从阳光中获取能量。这是食物网中能量流动的一部分，因为植物是生产者，食草动物可能会与它们相互作用。
  - **施肥和害虫控制**：为了确保农作物有足够的营养生长，可以施用化肥。可以使用杀虫剂或有机方法来控制可能损害农作物的害虫。这些方法可能通过影响某些物种对自然食物网产生影响，进而对生态系统产生连锁反应。
  - **收获**：当农作物成熟时，它们被收获，这暂时将植物从食物网中移除。
  - **收获后处理**：农作物可以被加工成食品，包装并运输到市场，这将农业周期与更广泛的经济联系起来。
  - **消费**：最后，农作物被人类和动物消费，或被转化为其他产品。
  - **分解**：任何有机废物可能会分解，将营养返回土壤。
- **蝙蝠**：作为对农业食物网有益的物种的一个例子，我们介绍了蝙蝠的重要性。它们在维持健康的生态系统中发挥着关键作用，这反过来又带来了显著的经济效益。这包括害虫控制、授粉和种子传播、生态系统平衡、粪便作为肥料、生态旅游，以及降低医疗保健成本。
- **边缘栖息地**：用于农业的田地周围的栖息地，作为田地与周围生态系统之间的缓冲区。











# **F：网络安全强大吗**

**背景**：随着现代技术的奇迹，我们的世界越来越多地通过互联网连接在一起。尽管这种在线连接性提高了全球生产力并让世界变得更小，但它也通过网络犯罪增加了我们个人和集体的脆弱性。网络犯罪难以应对，原因多种多样。许多网络安全事件跨越国界，使得调查和起诉这些犯罪的司法管辖问题变得复杂。此外，许多机构（如投资公司）不愿意报告被黑客攻击的情况，更倾向于悄悄支付赎金，而不是让客户和潜在客户知道他们遭受了安全漏洞。为了应对网络犯罪日益增长的成本和风险，许多国家已经制定了国家网络安全政策，这些政策可在其政府网站上公开获取。国际电信联盟（ITU）是联合国专注于信息和通信技术的专业机构；因此，它在制定国际标准、促进国际合作以及开发评估工具以帮助衡量全球和国家网络安全状况方面发挥着主导作用。

- 当你探索各国公布的国家安全政策，并将其与网络犯罪的分布进行比较时，出现了哪些模式，这些模式可以帮助你识别政策或法律中特别有效（或特别无效）的部分，以应对网络犯罪（通过预防、起诉或其他缓解措施）？根据你的分析方法，考虑每个政策被采纳的时间可能也是相关的。
