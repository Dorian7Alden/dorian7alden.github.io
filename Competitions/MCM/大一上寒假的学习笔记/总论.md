# 数学模型问题分类

- 优化类
- 评价类
- 预测类



**优化类问题**
1.1 数学规划模型
适用情况：要求一些变量有限制后，某一个变量或结果最大或最小，一般是最值问题

包括：线性规划、[整数线性规划](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=整数线性规划&zhida_source=entity)、非线性规划、多目标规划、[动态规划](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=动态规划&zhida_source=entity)。
（1）线性规划模型：目标函数和约束条件都是线性函数的数学规划模型；
（2）[整数规划](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=整数规划&zhida_source=entity)模型：决策变量要求取整数值的[线性规划模型](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=2&q=线性规划模型&zhida_source=entity)；
（3）非线性规划模型：目标函数或者约束条件中有[非线性函数](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=非线性函数&zhida_source=entity)的数学规划模型；
（4）多目标规划模型：具有多个目标函数的数学规划模型

1.2 [微分方程组](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=微分方程组&zhida_source=entity)模型
经常适用于人口预测，求最值极值点则为优化的目标

1.3 图论算法
最短路径问题、网络最大流问题、[最小费用最大流问题](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=最小费用最大流问题&zhida_source=entity)、最小生成树问题(MST)、[旅行商问题](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=旅行商问题&zhida_source=entity)(TSP)、图的着色问题





**评价类问题**
2.1 [层次分析法](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=层次分析法&zhida_source=entity)

是最基本的评价问题分析方法，能够求出某个因素的权重。以看出指标的影响力。有权重，用层次分析，分析指标
主观性比较强，要自己定指标，所以一般是辅助问题。

2.2 TOPSIS法
TOPSIS法根据有限个评价对象与理想化目标的接近程度进行排序的方法，是在现有的对象中进行相对优劣的评价。
可以加入熵权法分析权重


2.3 [主成分分析](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=主成分分析&zhida_source=entity)
在实际问题中，研究多指标(变量)问题是经常遇到的，然而在多数情况下，不同指标之间是有一定相关性。由于指标较多并且指标之间有一定的相关性，势必增加了分析问题的复杂性。
[数据降维](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=数据降维&zhida_source=entity)，方便计算，指标太多，减少点
主成分分析就是设法将原来众多具有一定相关性的指标(比如p个指标)，重新组合成一组新的相互无关的[综合指标](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=综合指标&zhida_source=entity)来代替原来指标。通常数学上的处理就是将原来p个指标作[线性组合](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=线性组合&zhida_source=entity)，作为新的综合指标

2.4 模糊评价
运用[模糊综合评价法](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=模糊综合评价法&zhida_source=entity)可以有效的 将一些定性指标进行定量化分析
比如，A大坝建设的过程中融资可能出现多个风险因素，而每个风险因素对A大坝的影响程度无法进行精准的量化评估，具有一定的模糊特征。基于此，采用模糊综合评价法对A大坝融资过程中可能发生的不确定性进行量化评估



**预测类问题**
般的抽象系统，如社会系统、经济系统、农业系统、生态系统等都包含有众多的因素来共同决定了该系统的发展态势，所以系统分析就是要找出哪些是主要因素，那些事次要因素，那些因素对系统发展的影响最大，那些因素对系统发展的影响最小，那些因素对系统发展起推动作用不要强化发展，哪些因素对系统发展起阻碍作用要加以抑制

3.1 [回归分析](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=回归分析&zhida_source=entity)
当样本个数n较大时，一般使用标准化回归,做[数据拟合](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=数据拟合&zhida_source=entity)，算参数

3.2 微分方程预测
一般适用于短中期预测。
例如：
传染病的[预测模型](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=预测模型&zhida_source=entity)、经济增长（或人口）的预测模型、Lanchester战争预测模型、药物在体内的分布与排除预测模型、烟雾的扩散与消失模型

构建方程，拟合参数

3.3 主成分分析
3.4 [灰色关联分析](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=灰色关联分析&zhida_source=entity)
作用：给出各个指标和结果，算权重，得到哪个影响力大。
可以是预测也可以是评价某个指标
当样本数n较少时，才使用灰色关联分析。

最后得出的[灰色关联度](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=灰色关联度&zhida_source=entity)在经过归一化后即为各个指标的权重。
到这一步得出的权重，谁的值大，那么哪个指标对系统的影响就大

3.4 元胞自动机
一般以分子的方式模拟一些情况的变化。有益于当题目中分析不到指标，或者指标太过于抽象和学术时，以分子的形式创建指标。

3.5 神经网络模型 [贝叶斯预测模型](https://zhida.zhihu.com/search?content_id=185837051&content_type=Article&match_order=1&q=贝叶斯预测模型&zhida_source=entity)
小样本情况用贝叶斯预测模型
大样本用神经网络模型比较准确
